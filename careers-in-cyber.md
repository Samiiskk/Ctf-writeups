## Why Cybersecurity?
- 3.5 million unfilled cybersecurity jobs globally
- High demand, competitive salaries
- Multiple career paths available

## Career Paths Overview
| Role | Team | Main Goal |
|------|------|----------|
| Security Analyst | Blue Team | Detect and investigate threats |
| Security Engineer | Blue Team | Build and maintain defenses |
| Penetration Tester | Red Team | Find vulnerabilities before attackers |

## Security Analyst (My Target Role)
- Monitor security alerts
- Investigate suspicious logins and activity
- Analyze logs and events
- Respond to incidents
- Improve security defenses

**Career path:**
Security Analyst → Threat Hunter → Incident Responder → SOC Lead

## Key Takeaway
- **Blue Team** = Monitor → Detect → Investigate → Respond
- **Security Engineer** = Build → Configure → Protect
- **Red Team** = Test → Exploit → Report
- Penetration testing requires an authorized **Engagement** agreement
